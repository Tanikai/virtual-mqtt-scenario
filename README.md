### Built With

* [Boost](https://www.boost.org/)

### Prerequisites

To build virtual-mqtt, you need the following software / libraries:

* [CMake](https://cmake.org/download/)
* Boost

If you have problems compiling or linking the Project with Boost, I can recommend [MinGW with bundled Boost by Stephan T. Lavavej](https://nuwen.net/mingw.html)

## Getting Started

To get a local copy up and running follow these simple example steps.

1. Clone the repo

   ```sh
   git clone https://github.com/Tanikai/virtual-mqtt.git
   cd virtual-mqtt
   ```

2. Initialize CMake:

   ```sh
   mkdir build && cd build
   cmake ..
   make
   ```
